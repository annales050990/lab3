3. Lista definicji – Podstawowe terminy HTML

<!DOCTYPE>
Deklaracja typu dokumentu – informuje przeglądarkę, w jakiej wersji HTML napisano stronę.
Tag
Podstawowy element składni HTML używany do tworzenia struktury dokumentu.
Atrybut
Dodatkowa informacja o tagu, np. class lub id.
Element
Tag otwierający, zawartość i tag zamykający – np. <p>Tekst</p>.
Hiperłącze
Odnośnik do innego dokumentu lub strony, tworzony za pomocą tagu <a>.
